**Opal Product Collection for WooCommerce** plugin for WordPress, an innovative tool designed to streamline content organization and product management. This plugin empowers users to create and manage collections effortlessly within their WordPress environment.

== Description ==
The primary function of the **Opal Product Collection for WooCommerce** is to facilitate the creation of curated collections, each tailored to specific criteria and conditions. Through an intuitive interface, users can define rules and parameters that automatically populate their collections with products that meet those criteria. This feature greatly enhances efficiency by automating the process of gathering relevant items, saving users valuable time and effort.

Moreover, the plugin offers versatile settings that allow users to extend its functionality beyond the default taxonomy. With the ability to integrate module selection for conditions and scanning across other taxonomies such as Product Categories and Product Tags, users can leverage the power of Collections Taxonomy across various aspects of their WordPress site, ensuring comprehensive content organization and management.

= Features =

* Create Collection taxonomy
* Module condition filter product for each collection
* Can choose products including or excluded
* Logo image for collection
* Scan products that meet the conditions and attach to the Collection
* Scan Schedule
* Can change collection slug
* Display "More in Collection" in product page
* Display "Collections" in metabox of product'summary
* Show SEO Meta

= Benefits:
* Save a lot of time and effort for administrators
* Improves sales performance and enhances the user experience.
* Helps optimize and be friendly to search engines

In essence, the **Opal Product Collection for WooCommerce** plugin revolutionizes the way users curate and manage content within WordPress, offering a seamless and efficient solution for creating and maintaining collections tailored to their specific needs and preferences.

== Installation ==
1. Unzip the downloaded zip file.
2. Upload the plugin folder into the wp-content/plugins/' directory of your WordPress site.
3. Activate plugin '**Opal Product Collection for WooCommerce**' from Plugins page

== Frequently Asked Questions ==

= Can I use multiple rules for a collection? =
Of course. You can add multiple conditions for each collection
= Does the plugin automatically scan for new products that match the collection's conditions? =
We provide settings that allow administrators to configure collections to automatically scan at specified times. Go to Woocommerce => Collection Settings => Scan Schedule => Active
